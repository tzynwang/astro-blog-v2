Display draft post

2021-02-19 14:25 hexo
  • Environment
  • Reference

To view the draft posts (in the source\_drafts folder) on localhost:4000, just run hexo server --draft. Set render_drafts as true in _config.yml will make hexo generate generate the posts in source\_drafts into public folder, which means the draft posts will be deployed to GitHub page by hexo deploy. No needs to set render_drafts as true to view the draft posts on localhost, running hexo server --draft is enough.
